为什么这门课程？

Hermes、OpenClaw、OpenCode、Claude Code、Cursor Agent，本质都基于同一结构： 

LLM → Reasoning → Tool Calling → Agent Loop → Memory → UI

真正的核心不是模型，而是 Agent Runtime。
